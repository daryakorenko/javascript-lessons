CSS свойства

display: flex - Делает все элементы резиновыми, а не строчными или блочными.

flex-direction - Определяет направление дочерних элементов. Принимает значения: row, row-reverse, column, column-reverse.

order - Порядок отображения flex-элементов. Свойство определяет порядок отображения и расположения элементов внутри контейнера. Значения указыватся числами. По умолчанию элементы имеют order: 0; 

order: 2;

Выравнивание по главной оси: justify-content. Принимает значения: flex-start, flex-end, center, space-between, space-around. 

justify-content: flex-end;

Hello B
Hello C

Выравнивание по поперечной оси: align-items. Принимает значения: flex-start, flex-end, center, baseline, stretch. 

align-items: flex-end;
