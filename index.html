Теги от <h1> до <h6> (heading) создают заголовки разного уровня (размера). Как правило на странице только один заголовок первого уровня.

Тег <p> (paragraph) создает абзац.

Теги <b> (bold) и <strong> применяют к тексту полужирное начертание.

Теги <i> (italics) и <em> (emphasis) применяют к тексту курсивное начертание.

Тег <u> (underline) делает текст подчеркнутым. Тег <s> (strikethrough) делает тест зачернутым.

Теги <sub> (subscript) и <sup> (superscript) делают текст подстрочным или надстрочным.

Специальные символы нужно указывать с помощью HTML-кодов. Например: &lt; &amp; &gt; для символов <, & и > соответственно. lt значит less than - меньше чем. gt значит greater than - больше чем.

Тег <br> (break) вставляет
перенос строки. И не требует закрывающего тега.

Тег <blockquote> обозначает цитату. По умолчанию визуально никак не оформлен. (Оформлен только отступами со всех сторон)

Ето цитата

Тег <ul> (unordered list) и <ol> (ordered list) обозначают маркированный и нумерованный списки.
<li> (list item) обозначает элемент списка.

  • морковь
  • картофель
  1. хлебушек
  2. варенье

Тег <div> создает блочный элемент.
Тег <span> создает строчный элемент.

Блочный элемент всегда начинается с новой строки и всегда занимает всю строку целиком.

Блочный элемент 1
Блочный элемент 2
Строчный элемент 1 Строчный элемент 2

Тег <a> (anchor) позволяет создать ссылку на элемент на этой странице или на другую страницу в интернете (на этом сайте или на другом).

Тег <a> принимает аттрибуты href и target. Примеры:

  • <a href="#some_id">Текст ссылки</a> - ссылается на элемент на странице с id="some_id", # обозначает,что указан id.
  • href="http://yandex.ru" ссылается на страницу в интернете.
  • target="_blank" - откроет ссылку в новом окне.

Тег <img> позволяет вставить картинку на страницу. Тег img принимает аттрибуты src, alt, width и height. Тег <img> не имеет закрывающего тега.

  • src - URL изображения, аттрибут является обязателным.
  • alt - текст, который появится при наведении на изображение или если изображение не смогло загрузиться.
  • width и height - ширина и высота изображения, можно указать в пикселях (100) или процентах (100%).
<img alt="Изображение Мейбл" src="images/mabel.jpg">

Тег <pre> представляет собой предварительно отформатированный текст, который должен быть представлен точно так, как написано в HTML-файле. Удобно писать примеры кода, команд, ответов сервера или консоли.

Тег <table> создает таблицу.

Тег <tr> определяет строку таблицы (table row).

Тег <td> опредляет ячейку.

Тег <thead> опредляет заглавие таблицы.

Тег <tbody> опредляет тело таблицы.

Тег <th> опредляет заглавную ячейку (типа как будто она из thead). По умолчанию текст внутри будет полужирный и выровнен по-центру.

rowspan="X" - аттурибут тега td, предлагает ячейке занять Х строк.

Формы

Форма для примера















Тег <form> создает форму, то есть область документа с интерактивными элементами и возможностью отправить данные на сервер. Принимает аттрибуты action и method

Тег <button> создает кнопку.

Тег<input> создает один из видов интерактивных элементов для ввода данных. Закрывающий тег ему не нужен. Тип определяется значением аттрибута type.

  • type="text" - поле ввода текста. Позволяет так же применить аттрибуты minlength и maxlength ограничивающие кол-во символов.
  • type="number" - поле ввода числа. Позволяет так же применить аттрибуты min и max ограничивающие диапазон вводимого числа.
  • type="password" - поле ввода пароля, пароль отображается в виде скрытых за точками символов.
  • type="checkbox" - чекбокс, позволяет применить аттрибут checked.
  • type="radio" - радио-кнопка, позволяет применить аттрибут checked.
  • type="date" - поле ввода даты с выпадающим календарем.
  • type="datetime-local" - поле ввода даты с выпадающим календарем и времени.
  • type="color" - поле выбора цвета.
  • type="file" - поле выбора файла. Принимает аттрибуты multiple и accept. Например: accept=".jpg,.jpeg,.png,.gif"
  • type="button" и "submit" - создает кнопки.

Тег <label> создает ярлык, который можно привязать к какому-то input при помощи аттрибута for. Для этого значением аттрибута for нужно указать существующий id того элемента, который хочешь привязать.

Важные аттрибуты элемента input: type, placeholder, accept, checked, disabled, readonly, value, name, minlength, maxlength, min, max

Аттрибуты HTML-тегов

С помощью аттрибутов тегам можно задать дополнительные свойства или настройки. Аттрибуты указывыются в открывающем теге:
<h1 id="heading">
<тег аттрибут="значение аттрибута">

Существуют общие аттрибуты, которые можно добавить к любоиу тегу и спецефические, которые применимы только к определенным тегам.

Примеры общих аттрибутов:

  • class
  • id
  • style
  • title - подсказка, которая появляется при наведении на элемент

Селекторы

С помощью селекторов мы выбираем к каким элементам на странице будут применены правила/стили CSS. 

.purple - выберет все элементы с классом purple
#title - выберет все элементы с id='title"
h1 - выберет все заголовки первого уровня
a[target="_blank"] - выберет все ссылки с аттрибутом target равным "_blank"

CSS-стили можно записать как в отдельный файл, так и в аттрибут конкретного элемента.

Размеры и отступы

width - задает ширину элемента.
height - задает высоту элемента.
Высоту и ширину нельзя применить к строчным элементам.
Принимает значения в px, %, vh, vw, mm, em и rem.

vh (viewport height) и vw (viewport width) - специальные единицы измерения. Viewport - это текущее окно браузера.
1vh - равен 1% от высоты окна. vw - то же самое с шириной.

min-width - задает минимальную ширину элемента.
min-height - задает минимальную высоту элемента.
Это значит, что ни при каких условиях элемент не будет сжат ниже этих значений.

max-width - задает максимальную ширину элемента.
max-height - задает максимальную высоту элемента.
Это значит, что ни при каких условиях элемент не будет растянут выше этих значений.

padding - отбивка/отступ внутрь от границ элемента.
margin - отбивка/отступ наружу от границ элемента. 

// варианты записи margin/padding
margin: 8px 12px 14px 16px; // ↑сверху →справа ↓снизу ←слева
margin: 8px 12px; // ↑↓сверху и снизу | справа и →←слева
margin: 8px; // со всех сторон одинаково

margin-top: 10px; // для каждой стороны отдельно
margin-right: 10px;
margin-bottom: 10px;
margin-left: 10px;

box-sizing принимает значения: border-box или content-box
border-box - к указанной ширине и/или высоте не добавлять значения padding и/или border.
content-box - к указанной ширине и/или высоте добавлять значения padding и/или border.

Фон и границы

border - задает рамку вокруг элемента.
background-color - задает цвет фона элемента.
border-radius - позволяет скруглять углы у элементов. 

border: 1px solid black; // запись в одно свойство

// запись в отдельных свойствах
border-width: 2px;
border-style: solid; // dotted, dashed, double, none
border-color: orange;

// запись для отдельных сторон
border-left: 1px solid black;

background-image -задает фоновое изображение для элемента.
background-repeat - повторение фоновой картинки. Принимает: no-repeat, repeat, repeat-x, repeat-y.
background-size - позволяет задать размер изображению. Указывается обычно в px, %.

background-image: url('http://ya.ru/logo.png');
background-image: url('images/cat.jpg');
background-repeat: repeat;
background-size: по-горизонтали по-вертикали;

overflow - позволяет решить что делать с содержимым, которое не влезло в границы элемента. Принимает значения:

  • visible - показывать
  • hidden - скрывать
  • auto - добавить полосу прокрутки если это нужно

Можно задать отдельно по горизонтали и вертикали: overflow-x, overflow-y

Текст и шрифт

color - задает цвет текста в элементе.

  • Английское название цвета: plum, white, salmon, red т.д.
  • hex-code - шестнадцатеричный цветовой код. Записывается через # с помощью 6 цифр/6 букв/буквы+цифры. Буквы используют только A B C D E F. Пример: # e7e7e7
  • rgb - еще один способ записи цвета, записывается (0, 0, 0)
  • rgba - способ записи прозрачности цвета, (0, 0 , 0, 1), первые три цифры-это цвет, а четвертая-прозрачность.

font-family - задает семейство шрифтов, принимает значения: serif (с засечками), sans-serif (без засечек) или названия семейств в кавычках через запятую. Если первое семейство не обнаружено, браузер попытается использовать следующее по списку.
font-size - задает размер шрифта. Принимает значения в px, %, mm, em и rem.
line-height - задает высоту строки, принимает те же значения.
font-weight - задает толщину начертания текста. 300 - тонкий, 400 - обычный, 500-900 - все жирнее и жирнее.

font-style - позволяет выбрать курсивное начертание если выбрано значение italic

text-align - горизонтальное выравнивание текста. Принимает значения: left, right, center и justify.

text-decoration - позволяет добавить или убрать подчеркиевание текста. Принимает значение: underline и none. Дополнительно через пробел можно указать цвет.

white-space - значение nowrap запрещает перенос строки внутри элемента.

Postman

pm.globals - обращается к списку глобальных переменных окружения (Environments -> Globals)

  • pm.globals.set("moya_peremennaya", "https://api.test.ru") - команда set() устанавливает значение глобальной переменной "moya_peremennaya" равное "https://api.test.ru"
  • pm.globals.get("moya_peremennaya") - команда get() возвращает значение глобальной переменной "moya_peremennaya"

pm.environment - обращается к текущему выбранному списку переменных окружения

  • pm.environment.set("moya_peremennaya", "https://api.test.ru") - команда set() устанавливает значение переменной "moya_peremennaya" равное "https://api.test.ru"
  • pm.environment.get("moya_peremennaya") - команда get() возвращает значение переменной "moya_peremennaya"

pm.response - обращается к ответу на запрос, который мы только что получили

  • pm.response.json() - вернет тело ответа (в формате JSON)
  • pm.response.status - вернет status ответа ("OK", "Bad Request", "Not Found")
  • pm.response.code - вернет code ответа (200, 400, 404)
  • pm.response.responseTime - вернет время ответа в миллисекундах
  • pm.response.headers - вернет headers ответа
  • pm.response.headers.get('Content-Type') - вернет header ответа, под названием Content-Type

pm.cookies - обращается к кукам

pm.test() - метод, запускающий автотест, первым аргументом принимает человекопонятное название теста, вторым - какое-то утверждение

pm.test("Статус ответа - ОК", () => {
  pm.expect(pm.response.status).to.eql("OK");
});

pm.test("Код ответа - 200", () => {
  pm.expect(pm.response.code).to.eql(200);
});

pm.test("Логин у пользователя - admin", () => {
  pm.expect(pm.response.json().login).to.eql("admin");
});

pm.expect()... - метод проверки ожиданий тестировщика от каких-либо значений ответа, составляется как предложение. Используется вместе с pm.test() - смотри примеры выше.

// pm.expect(...).to.eql(...) - строго равно
pm.expect(pm.response.json().login).to.eql('admin');
pm.expect(pm.response.headers.get('Content-Type')).to.eql('application/json');
pm.expect(pm.response.json().name).to.eql(pm.environment.get("name")); - сравнение ответа с некоей переменной

// pm.expect(...).to.have(...) - строка, массив или объект содержат соотв. значение
pm.response.to.have.header("Content-Type"); - без expect
pm.response.to.have.status(400); - без expect

// pm.expect(...).to.be.oneOf([..., ...]) - является одним из массива значений
pm.expect(pm.response.code).to.be.oneOf([201,202]); - код ответа либо 201, либо 202

// pm.expect(...).to.be.true - является true
// pm.expect(...).to.be.null - является null
// pm.expect(...).to.be.undefined - является undefined
// pm.expect(...).to.be.an('object') - является объектом (или array, string, number)
  
// pm.expect(...).to.be.below(...) - меньше значения (above - если больше)
pm.expect(pm.response.responseTime).to.be.below(200); - время ответа в миллисекундах
